Процесс модификации .ebuild

 Разработка на C# под linux
 
1) взять старую версию и скопировать её из основного дерева в оверлей
2) закоммитить
3) определить название новой версии .ebuild
4) переименовать .ebuild
5) закоммитить не внося других изменений (чтобы git видел что было именно переименование)
6) внести изменения
7) проверить устанавливаемость при помощи emerge
8) проверить репозиторий при помощи repoman -d
9) закоммитить
10) создать PR

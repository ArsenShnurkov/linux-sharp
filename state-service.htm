ASP.NET state service

 Как поместить идентификатор сессии в параметр GET, а не в URL?, Разработка на C# под linux
 
это такой сервис, который используется для хранения состояния сессий вместо SQL-сервера.
Что является его реализацией в mono?

The ASP.NET state service is installed with ASP.NET, and
by default is located in
%windir%\Microsoft.NET\Framework\version\aspnet_state.exe.

Как включить этот режим

In the Web.config file for the application, set
mode to "StateServer" and
stateConnectionString to a value, such as "tcpip=dataserver:42424".

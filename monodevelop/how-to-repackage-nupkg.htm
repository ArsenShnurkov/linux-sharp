Как быстро описать пакет .nupkg для gentoo?

 Как собирать monodevelop
 
сначала нужно скачать архив,
из архива извлечь .nuspec
из nuspec выписать зависимости и обработать их
создать файл ebuild с правильным именем (которое включает в себя версию)
вписать в ebuild имя пакета
со страницы взять DESCRIPTION и вписать его в ebuild
исправить файл metadata.xml
выполнить ebuild digest (потому что название иконки - как у пакета, то есть для разных пакетов названия иконок разные)
